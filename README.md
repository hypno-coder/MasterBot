# Запуск проекта локально:

1. **Активируем venv**
2. **Устанавливаем свой токен для бота в `.env BOT_TOKEN`**
3. **Запускаем Redis:** `redis-server`
4. **Запускаем Ngrok:** `tuna http 8000`
5. **В появившемся окне Ngrok в разделе Forwarding нужно скопировать ссылку слева от стрелки**
6. **Заменить эту ссылку в `main_dev.py` в переменной `WEBHOOK_NGROK`**
7. **После этого запускаем проект командой** `python main_dev.py`
